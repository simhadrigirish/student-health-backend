# Student Health & Wellness Backend

## Project Description
Student Health & Wellness is a Spring Boot backend project that provides health resources, emergency contacts, and student feedback management.

## Technologies Used
- Spring Boot
- MySQL
- JPA / Hibernate
- REST API
- Maven

## Features
- Student Registration
- Health Resources Management
- Emergency Contacts
- Feedback System

## API Endpoints

### Health Resources
GET /api/health  
POST /api/health  

### Student
POST /api/auth/register  
POST /api/auth/login  

### Emergency
GET /api/emergency  
POST /api/emergency  

### Feedback
GET /api/feedback  
POST /api/feedback  

## Run Project

1. Clone repository
2. Configure MySQL database
3. Run Spring Boot application

Backend runs on:
http://localhost:8083
